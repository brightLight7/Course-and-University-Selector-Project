Course & University Decision Support System

Final Year Project (FYP)
A web-based decision-support system to help international students choose suitable UK courses and universities.

Purpose
The project aims to reduce confusion and decision fatigue by providing:
- guided recommendations
- clear comparisons
- simple explanations

Features
- Short preference quiz
- Ranked course and university results
- Explanation of recommendations
- Mobile-friendly design
- Optional account saving

System Model
Input → Process → Output
User answers → Scoring logic → Ranked recommendations

Data & Ethics
- Uses a small, curated dataset
- No automated scraping
- No sensitive personal data
- Advisory recommendations only
